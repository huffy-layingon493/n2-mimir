## ✨ What n2-mimir does

n2-mimir helps an AI agent keep track of what it has learned from past experience. It does three main things:

- stores experience in a local database
- finds patterns in past runs
- turns those patterns into useful behavior changes

It uses TypeScript and Rust under the hood, with SQLite FTS5 for fast text search. It also supports semantic search, so the app can find related information even when the words do not match exactly.

## 🗂️ How it works

n2-mimir keeps a local record of experience. It can:

- collect events from an agent
- store those events in SQLite
- index text with FTS5 for quick search
- detect repeated patterns
- build insights from older runs
- use those insights to guide future behavior

This setup helps the app act less like a short-term memory buffer and more like a learning layer.

## 🔎 Search and memory

The app includes two ways to find past information:

- **Keyword search** for exact terms
- **Semantic search** for related meaning

That means you can look up a task by name, a phrase from a note, or a similar idea from an old run.

## 🧰 Troubleshooting

If the app does not start:

1. Download the latest release again.
2. Check that you chose the Windows file.
3. Right-click the file and choose **Run as administrator**.
4. Make sure Windows has not blocked the download.
5. Restart your computer and try again.
6. If the file is inside a ZIP folder, extract it first.

If search feels slow:

1. Wait for the first index step to finish.
2. Keep the database on a local drive.
3. Close other apps that use a lot of memory.

If the app does not remember data:

1. Check that the storage folder still exists.
2. Make sure you did not move or delete the database file.
3. Open the app with the same user account you used before.
